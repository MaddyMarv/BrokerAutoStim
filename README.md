### [Broker Auto Stim](https://www.nexusmods.com/warhammer40kdarktide/mods/655)
**The short version:** Automatically injects the broker stim for you after you've been in combat for a set time.

**The details:** Standalone mod that handles using your broker stims for you. It automatically pops the stim once you have been fighting for a few seconds. It has tons of customization. You can set up multiple profiles, cancel the injection animation automatically, and prevent it from firing while you are attacking, blocking, or doing interactions.
